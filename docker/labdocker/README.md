# labdocker — Services essentiels

VM hébergée sur Proxmox. Regroupe les services du quotidien : dashboard personnel, administration, monitoring, outils de conversion et synchronisation de repositories locaux.

## Services

**Homepage** — Dashboard personnalisé. Point d'entrée vers tous les services.

**Portainer** — Administration des conteneurs Docker via interface Web.

**Beszel** — Monitoring temps réel (CPU, RAM, réseau) pour tous les serveurs du homelab.

**Dozzle** — Consultation des logs Docker en direct.

**IT-Tools** — Boîte à outils développeur (encodeurs, décodeurs, générateurs).

**Stirling PDF** — Édition et manipulation de fichiers PDF.

**ConvertX** — Conversion de fichiers (images, vidéos, documents).

**Wealthfolio** — Suivi de portefeuille d'investissements.

**Gitea Mirror** — Mirror local des repositories pour redondance et accès hors ligne.

**Diun** — Notifications de mises à jour Docker.
