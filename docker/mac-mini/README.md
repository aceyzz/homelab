# mac-mini — Services GPU/CPU intensives

Machine dédiée aux services gourmands en ressources : inférence LLM locale, stockage collaboratif et monitoring système.

## Services

**Open WebUI** — Interface pour modèles LLM locaux (Llama, Mistral, Qwen, etc.). Chat et génération de texte sans dépendance cloud.

**Nextcloud** — Stockage en ligne self-hosted. Synchronisation de fichiers et édition collaborative via OnlyOffice.

**Dozzle** — Consultation des logs Docker en direct.

**Beszel Agent** — Envoie les données système (CPU, RAM, GPU, disque) au serveur Beszel centralisé.

**Diun** — Notifications de mises à jour Docker.
